# Ubuntu_Install_Tensorflow
Ubuntu 16.04安装Tensowflow框架记录

由于本科毕业设计有关深度学习，接触其有一段时间，在实验室电脑中配置Tensorflow的框架，由于我的笔记本电脑是XP系统，所以可能在配置的过程中会出现一些问题，实验室电脑配置：

**硬件：**
内存：7.7 GiB

处理器：i7

操作系统：64位

**软件：**
Ubuntu16.04

CUDA 8.0

Cudnn v5

Tensorflow 0.11

####1.1  CUDA 8.0
